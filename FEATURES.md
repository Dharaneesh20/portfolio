## 🔌 API Endpoints

### Projects
```
GET    /api/projects       - List all
GET    /api/projects/:id   - Get one
POST   /api/projects       - Create
PUT    /api/projects/:id   - Update
DELETE /api/projects/:id   - Delete
```

### Certifications
```
GET    /api/certifications       - List all
GET    /api/certifications/:id   - Get one
POST   /api/certifications       - Create
PUT    /api/certifications/:id   - Update
DELETE /api/certifications/:id   - Delete
```

### Blog
```
GET    /api/blog       - List all
GET    /api/blog/:id   - Get one
POST   /api/blog       - Create
PUT    /api/blog/:id   - Update
DELETE /api/blog/:id   - Delete
```

### CV
```
GET    /api/cv    - Get CV data
PUT    /api/cv    - Update CV
```

## 🗄️ Database Schema

### Project
```javascript
{
  title: String,
  description: String,
  technologies: [String],
  cloudProvider: String,
  image: String,
  githubUrl: String,
  liveUrl: String,
  status: 'completed' | 'ongoing' | 'upcoming'
}
```

### Certification
```javascript
{
  title: String,
  issuer: String,
  date: Date,
  cloudProvider: String,
  image: String,
  credentialUrl: String,
  description: String
}
```

### BlogPost
```javascript
{
  title: String,
  content: String,
  excerpt: String,
  author: String,
  date: Date,
  cloudProvider: String,
  image: String,
  tags: [String]
}
```

### CV
```javascript
{
  name: String,
  title: String,
  summary: String,
  experience: [Object],
  education: [Object],
  skills: [Object],
  achievements: [String]
}
```
